# Homebrew Tap for OpenCode Bar

## Install

```bash
brew tap kargnas/opencode-bar
brew install --cask opencode-bar
```

## What is OpenCode Bar?

A macOS menu bar app that monitors AI provider usage from [OpenCode](https://opencode.ai) in real-time.

Includes a CLI tool (`opencodebar`) for terminal usage.
